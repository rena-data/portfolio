## Stack

- **Frontend**: Vanilla HTML/CSS/JS (SPA, hash-based routing)
- **Hosting**: GitHub Pages
- **Font**: Pretendard Variable
- **Analytics**: GA4 (쿠키 동의 시에만)
- **Contact**: Google Sheets + Apps Script + Slack 알림
